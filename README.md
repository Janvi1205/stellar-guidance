# Astro Aarpit - Vedic Astrology Consultation

Unlock the secrets of your birth chart with personalized Vedic astrology consultations. Guidance for love, career, health & spiritual growth.

## Technologies Used

- **Framework**: [React](https://reactjs.org/) with [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Components**: [shadcn/ui](https://ui.shadcn.com/)
- **Animations**: [GSAP](https://greensock.com/gsap/) (GreenSock Animation Platform)
- **Icons**: [Lucide React](https://lucide.dev/)

## Core Services

- **Birth Chart Analysis**: Deep dive into your natal chart to uncover your life's blueprint.
- **Love & Relationships**: Compatibility analysis and timing guidance for meaningful connections.
- **Career & Finance**: Discover your professional destiny and optimal timing for decisions.
- **Vastu & Relocation**: Astrological guidance for home energy alignment.
- **Spiritual Growth**: Unlock your spiritual potential through planetary insights.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or later)
- npm or yarn

### Installation

1. Clone the repository:
   ```sh
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```sh
   cd stellar-guidance
   ```

3. Install dependencies:
   ```sh
   npm install
   ```

4. Start the development server:
   ```sh
   npm run dev
   ```
